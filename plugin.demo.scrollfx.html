PluginScrollFx Live Demo

Bind any CSS property to scroll position. The plugin writes a --scroll-fx-{property} CSS variable on the element as you scroll, interpolated linearly between two values across a viewport range you define. Use the variable in any CSS rule to drive parallax, fade, blur, rotation, color shifts — anything.

Each section below has data-scroll-fx attributes. Scroll the page — open DevTools and inspect any .js-scroll-fx element to watch its inline CSS variables update in real time.

How it works

Scroll to see each effect

Each section below is wired to data-scroll-fx attributes. The plugin writes a --scroll-fx-{property} CSS variable as you scroll; your CSS reads it. The page is intentionally tall — keep scrolling to trigger every stage.

fade + translate · combined

Fade-up on entry

The classic. As this card enters between 90% and 55% of the viewport, opacity ramps from 0 to 1 and Y translation eases from 60px to 0.

<section class="js-scroll-fx"
         data-scroll-fx-1="opacity,0,1,90,55"
         data-scroll-fx-2="translateY,60px,0px,90,55">...</section>

translateY · single axis

Parallax background

The image moves slower than the page. Range is wider (100→0) so the effect persists through the whole scroll.

<img class="js-scroll-fx"
     data-scroll-fx="translateY,-120px,120px,100,0">

opacity + translate + scale + rotate

Four effects, one element

Up to nine effects stack on a single element via data-scroll-fx-1… data-scroll-fx-N.

<div class="fx-multi js-scroll-fx"
     data-scroll-fx-1="opacity,0.2,1,100,50"
     data-scroll-fx-2="translateY,140px,0px,100,50"
     data-scroll-fx-3="scale,0.85,1,100,50"
     data-scroll-fx-4="rotate,-6deg,0deg,100,50">...</div>

filter: blur

Unblur as you scroll in

Heavy effect — reserve for hero sections.

<img class="js-scroll-fx" data-scroll-fx="blur,10px,0px,80,40">
/* CSS */ img { filter: blur(var(--scroll-fx-blur, 0)); }

rotate · scroll-driven

Rotate as the page moves

Useful for chevron pointers, decorative spinners, or logo flairs.

<div class="fx-rotate js-scroll-fx"
     data-scroll-fx="rotate,-180deg,180deg,100,0">...</div>

Color shifts by scroll position

Non-numeric properties (like text color via data-theme) are supported too — see the source comments. Here we drive filter: hue-rotate.
