Options reference

data-plugin-options / JS constructor options
Option Type Default Description
collapsed Boolean false Start in icon-only mini mode
activeTracking Boolean true Auto-detect active link from window.location.pathname
hashTracking Boolean false Also match location.hash when detecting active links
expandActive Boolean true Auto-open the group containing the active link on init
toggleBtn Boolean true Inject the hamburger/arrow toggle button
toggleTarget String null Selector for an additional element to receive .ts-vn-collapsed (e.g. the main content wrapper for a "push" layout)
animDuration Number 260 Group open/close slide animation in milliseconds
tooltips Boolean true Show Bootstrap 5 Tooltips on nav items when collapsed
Public API methods
MethodDescription
collapse()Switch to icon-only mini mode; fires verticalNav.collapsed event
expand()Switch to full-width mode; fires verticalNav.expanded event
toggle()Toggle between collapsed and expanded states
openGroup($group)Expand a .ts-vn-group jQuery element with slide animation
closeGroup($group)Collapse a .ts-vn-group jQuery element with slide animation
setActive(href)Programmatically move the active state to the link matching href
destroy()Tear down plugin, restore original markup, remove data key

themestrap.init.js wiring

// Vertical Navigation
if ($.isFunction($.fn['themestrapPluginVerticalNav']) && $('[data-plugin-vertical-nav]').length) {
    themestrap.fn.intObsInit('[data-plugin-vertical-nav]:not(.manual)', 'themestrapPluginVerticalNav');
}
<!-- Minimal dark nav -->
<nav data-plugin-vertical-nav
     data-plugin-options='{"collapsed": false}'>

  <div class="ts-vn-header">
    <span class="ts-vn-brand">My App</span>
  </div>

  <div class="ts-vn-body">

    <!-- Direct link -->
    <a href="/dashboard" class="ts-vn-link active">
      <span class="ts-vn-icon"><i class="bi bi-house-fill"></i></span>
      <span class="ts-vn-text">Dashboard</span>
    </a>

    <!-- Expandable group -->
    <div class="ts-vn-group">
      <button class="ts-vn-group-trigger">
        <span class="ts-vn-icon"><i class="bi bi-gear-fill"></i></span>
        <span class="ts-vn-text">Settings</span>
      </button>
      <div class="ts-vn-group-children">
        <a href="/settings/profile" class="ts-vn-link">
          <span class="ts-vn-text">Profile</span>
        </a>
      </div>
    </div>

  </div>
</nav>

<!-- Access instance via data key -->
<script>
  const nav = $('[data-plugin-vertical-nav]').data('__verticalNav');
  nav.collapse();
</script>
