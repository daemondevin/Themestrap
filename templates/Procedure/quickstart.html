Quickstart

Install a client library, authenticate, and make your first request.

Install the SDK

Add the official client to your project. Every SDK wraps the same REST endpoints documented in the API reference.

npm install @protocol/sdk
pip install protocol
go get github.com/protocol/protocol-go

Make a request

Instantiate a client with your secret key and call a resource method:

import Procedure from '@protocol/sdk'

const client = new Procedure({ token: process.env.PROTOCOL_KEY })
const { data } = await client.messages.list({ limit: 10 })

console.log(data)
import os, protocol

client = protocol.Client(token=os.environ['PROTOCOL_KEY'])
messages = client.messages.list(limit=10)

print(messages.data)

Authenticate

All requests must carry a bearer token in the Authorization header. See Authentication for key rotation and scopes.

curl https://api.protocol.dev/v1/messages \
  -H "Authorization: Bearer {token}"
Keep keys server-sideSecret keys grant full account access. Use a publishable key for browser code, and proxy privileged calls through your backend.
