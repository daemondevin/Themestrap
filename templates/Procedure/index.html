API reference

API documentation, assembled from the toolkit.

A demo website inspired by Tailwind's Protocol API reference template built with the following Themestrap plugins: CommandMenu, Highlight, CodeWindow, DarkMode, Alert, ScrollSpy, ScrollShadow, and the reason for this demo — CodeRail.

Welcome to the Procedure API reference — a complete documentation site built without a single bespoke widget. Search runs on the Command Menu, every snippet is rendered by Highlight inside a Code Window, the theme toggle is Dark Mode, the left rail tracks your scroll with Scroll Spy, and each endpoint page uses the new Code Rail two-column layout.

curl https://api.protocol.dev/v1/messages \
  -H "Authorization: Bearer {token}"
import protocol

client = protocol.Client(token="...")
messages = client.messages.list(limit=10)
import Procedure from '@protocol/sdk'

const client = new Procedure({ token: '...' })
const messages = await client.messages.list({ limit: 10 })

Get started

Install a client library and authenticate with your secret key:

npm install @protocol/sdk
# or
pip install protocol
Generate a secret key in the dashboard, then store it in an environment variable — never commit it to source control.
