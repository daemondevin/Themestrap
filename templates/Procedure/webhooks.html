Webhooks

Receive real-time events at your own endpoint, and verify that each delivery genuinely came from Procedure.

Register an endpoint

Create a webhook endpoint pointing at a publicly reachable URL on your server, and subscribe it to the events you care about.

curl https://api.protocol.dev/v1/webhooks \
  -H "Authorization: Bearer {token}" \
  -d url="https://example.com/hooks" \
  -d events[]="message.created"

Event payloads

Each delivery is a POST with an event envelope wrapping the resource:

{
  "id": "evt_5Lp2",
  "type": "message.created",
  "data": { "id": "msg_2Hk9", "object": "message" }
}

Verify signatures

Every request carries a Procedure-Signature header. Recompute the HMAC with your signing secret and compare in constant time:

import crypto from 'crypto'

function verify(body, sig, secret) {
  const digest = crypto.createHmac('sha256', secret)
    .update(body).digest('hex')
  return crypto.timingSafeEqual(Buffer.from(sig), Buffer.from(digest))
}
import hmac, hashlib

def verify(body, sig, secret):
    digest = hmac.new(secret.encode(), body, hashlib.sha256).hexdigest()
    return hmac.compare_digest(sig, digest)

Acknowledge with a 2xx within a few seconds and do heavy work asynchronously — we retry deliveries that time out.
