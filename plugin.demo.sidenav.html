PluginSideNav — Themestrap

A sidenav built on Themestrap's jQuery plugin architecture. Supports collapsible groups, a mini (icon-only) mode, dark variant, active state tracking, tooltip labels when collapsed, and a full public API.

HTML Usage

<nav data-plugin-sidenav
     data-plugin-options='{"collapsed": false, "dark": false, "showToggle": true}'>

  <div data-sidenav-header>
    <div data-sidenav-logo><i class="bi bi-layers-fill"></i></div>
    <span data-sidenav-title>My App</span>
  </div>

  <div data-sidenav-body>
    <div data-sidenav-group data-sidenav-group-title="Main">

      <!-- Simple item -->
      <a href="/dashboard" data-sidenav-item data-sidenav-active>
        <i class="bi bi-house" data-sidenav-icon></i>
        <span data-sidenav-label>Dashboard</span>
        <span data-sidenav-badge>3</span>
      </a>

      <!-- Item with nested sub-nav -->
      <div data-sidenav-item data-sidenav-has-children>
        <i class="bi bi-gear" data-sidenav-icon></i>
        <span data-sidenav-label>Settings</span>
        <span data-sidenav-chevron></span>
        <div data-sidenav-sub-items>
          <a href="/settings/general" data-sidenav-sub-item>General</a>
          <a href="/settings/api" data-sidenav-sub-item>API Keys</a>
        </div>
      </div>

    </div>
  </div>

  <div data-sidenav-footer>
    <a href="/logout" data-sidenav-item>
      <i class="bi bi-box-arrow-right" data-sidenav-icon></i>
      <span data-sidenav-label>Sign Out</span>
    </a>
  </div>

</nav>

JS API

// Access instance
const nav = $('#mySideNav').data('__pluginAppSideNav');

// Or via jQuery method (also inits if not yet inited)
const nav = $('#mySideNav').themestrapPluginAppSideNav({ dark: true });

// Methods
nav.collapse();          // icon-only mode
nav.expand();            // full width
nav.toggle();            // flip between the two

nav.setActive($item);    // set a .ts-asn__item as active
nav.setSubActive($sub);  // set a .ts-asn__sub-item as active (also marks parent)

nav.destroy();           // full teardown, restores original HTML

// Events (fired on the root <nav> element)
$('#mySideNav').on('item.ts.appsidenav',       (e, data) => { /* data.$item, data.href  */ });
$('#mySideNav').on('subitem.ts.appsidenav',    (e, data) => { /* data.$item, data.href  */ });
$('#mySideNav').on('toggle.ts.appsidenav',     (e, data) => { /* data.collapsed          */ });
$('#mySideNav').on('group-toggle.ts.appsidenav',(e, data) => { /* data.$item, data.open  */ });

Options

Option Type Default Description
collapsed boolean false Start in icon-only mode.
dark boolean false Use dark color scheme.
width string '260px' Expanded sidebar width.
widthCollapsed string '64px' Collapsed sidebar width.
duration string '250ms' CSS transition duration.
showToggle boolean true Inject collapse toggle button into the header.
activeOnLoad boolean true Auto-match active item from current URL.
autoCollapse boolean false Collapse sidebar when a leaf item is clicked.
