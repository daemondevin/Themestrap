Themestrap · Plugin

Context Menu

A right-click context menu with sub-menus, separators, keyboard navigation, dark mode, and smart viewport positioning with zero dependencies beyond jQuery.

Basic Menu

right-click
Document Area Right-click to open menu
HTML
<!-- Trigger element -->
<div id="my-area"
     data-plugin-context-menu
     data-plugin-options='{"items": [...]}'>
  Right-click me
</div>
JavaScript
// Auto-init via data attribute + init.js,
// or manual init:
$('#my-area').themestrapPluginContextMenu({
  items: [
    { type: 'label', text: 'File' },
    { label: 'New File',  icon: '📄', shortcut: '⌘N',
      action: () => console.log('new') },
    { label: 'Open…',    icon: '📂', shortcut: '⌘O',
      action: () => console.log('open') },
    { type: 'separator' },
    { label: 'Delete', icon: '🗑', destructive: true,
      action: () => console.log('delete') }
  ]
});

Dynamic Items + Callbacks

setItems()
Dynamic Menu Right-click · Use controls to swap items
JavaScript
// Initial items
const viewItems = [
  { label: 'Preview', icon: '👁', action: () => {} },
  { label: 'Full screen', icon: '⛶', action: () => {} }
];
const editItems = [
  { label: 'Cut',   icon: '✂️', action: () => {} },
  { label: 'Copy',  icon: '📋', action: () => {} },
  { label: 'Paste', icon: '📌', action: () => {} }
];

// Init with callbacks
$('#ctx-dynamic').themestrapPluginContextMenu({
  items: viewItems,
  onOpen:  (inst) => logEvent('ts-ctx-open'),
  onClose: (inst) => logEvent('ts-ctx-close')
});

// Swap items at runtime
$('#btn-mode-edit').on('click', () => {
  $('#ctx-dynamic').data('__pluginContextMenu').setItems(editItems);
});

Options Reference

OptionTypeDefaultDescription
itemsArray[]Array of item objects. See item schema below.
darkBooleanfalseApply dark theme to the menu container.
ariaLabelString'Context menu'aria-label on the menu element.
triggerEventString|nullnullSet 'longpress' to also open on mobile touch-and-hold.
longpressDelayNumber600Milliseconds before longpress triggers the menu.
mutualExclusionBooleantrueOpening one menu closes all other open context menus.
onOpenFunction|nullnullCallback fired when menu opens. Receives (instance).
onCloseFunction|nullnullCallback fired when menu closes. Receives (instance).

Item schema

KeyTypeDescription
labelStringItem display text.
actionFunctionClick handler. Receives (event, item). Menu closes after.
iconStringHTML string (emoji or SVG) rendered left of the label.
shortcutStringKeyboard shortcut hint displayed on the right.
disabledBooleanRenders item as non-interactive.
destructiveBooleanRenders item in red (danger colour).
itemsArrayNest a sub-menu. Hover or → key to open.
type: 'separator'—Horizontal divider line.
type: 'label' + text—Non-interactive group label.
