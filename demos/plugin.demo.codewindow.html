JS Plugin

PluginCodeWindow

A macOS-style window around your <pre> code blocks — traffic-light chrome, a filename tab strip, an ambient glow, and a real ARIA tab pattern. Highlighting is delegated to PluginHighlight. Full reference on the documentation page.

The default Windows chrome signature

A multi-tab window

Each <pre> becomes a tab. Click them, or focus one and use the arrow keys.

// themestrap.config.js
export default {
  theme: 'dark',
  plugins: {
    commandMenu: { shortcut: 'mod+k' },
    highlight:   { lineNumbers: true },
    darkMode:    { storageKey: 'themestrap-theme' },
  },
}
import themestrap from './themestrap.js';

themestrap.boot();
console.log('themestrap ready');

One pane, glow off

One <pre> adds the --single modifier; the tab underline is hidden.

return function (array $config): string {
    return $config['theme'] ?? 'light';
