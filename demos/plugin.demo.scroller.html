A native, dependency-free custom scrollbar — a from-scratch alternative to nanoScroller. The platform scrollbar is hidden and a themable, draggable bar is drawn on top. Drag the thumb, click the track to page, or scroll normally. No third-party JavaScript.

height: 320 · auto-hide

Fixed-height panel

A constrained host with overflowing prose. The bar fades in on scroll or hover, then auto-hides after a moment.

Why a custom scrollbar?

Native scrollbars are inconsistent across platforms and nearly impossible to style to a brand. PluginScroller hides the native bar and renders its own, so the look is identical everywhere and themed through a handful of CSS custom properties.

The content keeps scrolling natively — wheel, trackpad, keyboard and touch all work as usual — because the plugin never intercepts the scroll itself. It simply mirrors the native scroll position onto a thumb, and mirrors thumb drags back onto the native scroll. One source of truth, no jank.

How the thumb is sized

The thumb height is proportional to the visible-to-total ratio: a viewport showing a third of its content gets a thumb roughly a third of the track tall, clamped to a sensible minimum so it stays grabbable on very long pages.

When the container or its content changes size, a ResizeObserver and MutationObserver recalculate the geometry automatically. You rarely need to call update() yourself.

Theming

Every visual aspect — width, colour, hover colour, corner radius, inset — is a CSS variable on the host, so you can restyle one container or every scroller on the site from a stylesheet without touching JavaScript.

That is the whole idea: behaviour in options, looks in CSS.

<div data-plugin-scroller data-plugin-options='{"height":320}'>...</div>

// themestrap.plugin.scroller.js (excerpt)
update() {
    const viewportH = content.clientHeight;
    const contentH  = content.scrollHeight;
    const barH      = this.$bar[0].clientHeight;

    const hasOverflow = contentH > viewportH + 1;
    this.$el.toggleClass('ts-scroller--has-overflow', hasOverflow);
    if (!hasOverflow) return this;

    let thumbH = Math.round(barH * (viewportH / contentH));
    thumbH = Math.max(this.options.sliderMinHeight, thumbH);
    thumbH = Math.min(thumbH, barH);

    this._maxScroll      = contentH - viewportH;
    this._maxThumbTravel = Math.max(0, barH - thumbH);
    this.$thumb.css('height', thumbH + 'px');
    this._positionThumb();
    return this;
}

_positionThumb() {
    const ratio = this._maxScroll > 0
        ? this.$content[0].scrollTop / this._maxScroll : 0;
    const y = ratio * this._maxThumbTravel;
    this.$thumb.css('transform', 'translateY(' + y + 'px)');
}
