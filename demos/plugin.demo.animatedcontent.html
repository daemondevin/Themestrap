PluginAnimatedContent Live Demo

Splits a heading into individual letters (or words), wraps each fragment in a <span>, and plays an animate.css animation across them in sequence with a configurable stagger. Great for hero titles and section headings — the kind of motion that makes a page feel alive without being noisy.

Each example below uses the real plugin. The replay buttons call destroy() on the instance then re-initialize it. The bottom card swaps the animationName live.

letter · animationName: fadeIn (default)

Each character animates independently

The default split granularity is letter. The plugin wraps each character in a span and applies an animation with a per-letter delay equal to animationSpeed ms × index.
