Plugin

PluginMarkdown

Interactive demos for Themestrap's zero-dependency GFM Markdown parser. Add data-plugin-markdown to any element to render its text content — or point data-plugin-markdown-src at a .md file URL. All demos are live — edit, toggle, and re-render in the browser.

API

Programmatic API

methods

Use the controls below to call instance methods…

Click a button to run a method call… 
// Init
const md = $('#demo-api-output').themestrapPluginMarkdown()
              .data('__pluginMarkdown');

// Render a Markdown string
md.render('# Hello World\n\nSome **bold** text.');

// Get the last rendered HTML
const html = md.getHtml();

// Destroy and restore original content
md.destroy();
