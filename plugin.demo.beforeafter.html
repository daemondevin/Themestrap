PluginBeforeAfter Live Demo

A draggable image-comparison slider — the classic before/after reveal. Stack two same-size images inside a wrapper, attach the plugin, and a handle lets visitors wipe between them. Horizontal or vertical, with custom labels and interaction modes.

In production the plugin wraps the TwentyTwenty library shipped inside plugins.min.js. This demo bundles a minimal vanilla implementation so the page works standalone — the public API is identical.

orientation: horizontal · default

Drag the handle

The canonical pattern: a horizontal handle wipes between two stacked images.

<div data-plugin-before-after>
  <img src="before.jpg">
  <img src="after.jpg">
</div>

orientation: vertical

Top / bottom wipe

Useful when the difference between images is concentrated vertically — exteriors vs interiors, day vs night skies.

<div data-plugin-before-after
     data-plugin-options='{"orientation":"vertical"}'>...</div>

move_slider_on_hover · no_overlay

Hover-driven, label-free

No drag required — the handle follows the cursor. The overlay labels are hidden via no_overlay.

before_label · after_label · default_offset_pct

Custom labels, off-center handle

Initial position is shifted to 25% with default_offset_pct: 0.25. Hover anywhere on the image to see the captions.

click_to_move

Tap anywhere to set the divider

A precision mode for compact viewports. Each click jumps the handle to that exact horizontal position.

Options reference

What each flag does

  • orientation — horizontal (left/right wipe) or vertical (top/bottom).
  • default_offset_pct — Starting handle position as a fraction 0–1.
  • before_label · after_label — Captions shown on hover.
  • no_overlay — Hide hover labels entirely.
  • move_slider_on_hover — Divider follows the cursor without dragging.
  • move_with_handle_only — Restrict drag to the round handle.
  • click_to_move — Tap anywhere to jump the handle.
